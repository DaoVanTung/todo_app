## Lưu ý
Khi chạy trên thiết bị Android vật lý, cần chuyển hướng cổng 8000 của điện thoại đến cổng của máy tính. Chạy câu lệnh sau trên Terminal:
 - adb reverse tcp:8000 tcp:8000
